Atlanta Food Finder

Project Overview

Atlanta Food Finder is a web application designed to help users search for and discover restaurants in Atlanta. The application allows users to view restaurant locations on a map, read reviews, and get directions. It provides essential information such as restaurant names, addresses, cuisine types, ratings, and opening hours.

What Was Needed

The goal was to create a user-friendly interface that not only looks appealing but also delivers essential information to users efficiently. The project needed to incorporate:

  • Restaurant search functionality
  • Location-based maps for easy navigation
  • Rating and review systems for restaurants
  • Aesthetic and functional UI that reflects Atlanta's cultural vibe

Why the Design?

The UI was designed to reflect the vibrant Atlanta culture by using a black, red, and white color scheme, inspired by Atlanta's beloved Falcons and Braves teams. The sleek and simple design makes it easy to navigate, providing users with a seamless food-finding experience.

Hypertransic App

Project Overview

Hypertransic is a futuristic app designed to streamline rideshare options using advanced transportation systems like hoverboards, hyperloops, and flying cars. The app provides users with easy access to futuristic transit choices and allows seamless booking.

Virtual Pickup

In the future, geotracking will be a common feature, allowing the app to identify the user's location based on a single photo. This feature, known as "Virtual Pickup," enables users to simply take a photo, and the app will determine their location for a more convenient pickup.

Additional Features

To enhance the user experience, we added essential features such as a wallet for past transactions, a hyperloop schedule to see upcoming train arrivals, and a personalized profile page. These features help streamline future travel planning while keeping the user engaged and informed.

Semantic Safari

Project Overview

Semantic Safari is an engaging web game designed to help students learn in a fun and interactive way. The game uses a ChatGPT API to generate questions, with three random wrong answers and one correct answer. Players have one minute to answer as many questions as possible, gaining points with every correct answer and streaks for bonus points. The simple yet effective UI makes it easy for students to enjoy while learning.

Home Screen

Game in Progress

Players must answer as many questions as possible within the one-minute time frame. Streaks are rewarded with higher points, motivating players to maintain accuracy and speed. The game is fast-paced, making it exciting for students to learn through challenge.

Game in Progress

Game Wrap-up

After the one-minute countdown, players are shown their final score, based on the number of questions answered correctly and any streak bonuses. The final screen encourages replaying to improve scores and further engage with the learning process.
